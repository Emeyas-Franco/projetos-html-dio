História dos computadores

Primeira Geração (1944-1959)

Com suas enormes dimensões, os computadores eram usados por militares para calculos durante as guerras.

Segunda Geração (1959-1965)

Com a evolução das válvulas para os transistores. Os computadores ainda tiham grandes dimensões, porém já começaram a ter um uso comercial.

Terceira Geração (1965-1970)

Com a implementação de circuitos integrados teve uma redução considerável em seus tamanhos e começaram a ser utilizados como computadores pessoais.

Quarta Geração (a partir de 1971)

Com a inclusão de microprocessadores, os computadores estão cada vez menores, a partir dos anos 90 a expansão de computadores pessoais foi enorme.

O que são Clients?

Clients no linguajar web, o que é mostrado nos navegadores, atráves de páginas web, aplicações web, aplicativos entre outros, que são desenvolvido através de linguagens como HTML5, CSS3, JavaScript.

O que são Servers?

Servers são os servidores para armazenamento de informações e bancos de dados das páginas que são exibidas nas páginas web ou em aplicativos.

O que é uma estrutura básica de um HTML?

A estrutura básica de um HTML é baseado em tags:

  • HTML: tag usada para declarar o início de uma página HTML, o início de toda a estrutura, é obrigatório a abertura <html> e o fechamento desta tag </html>;
  • HEAD: Tag usada para declarar as informações gerais do documento, o que inclui o título e links para folhas de estilho e scripts,é obrigatório a abertura <head> e o fechamento desta tag </head>;
  • BODY: Tag usada para declarar o corpo da página, todo o conteúdo como (textos, imagens e vídeos)que será disponibilizado através de tags, convertidas em conteúdo web para o usuário, é obrigatório a abertura <body> e o fechamento desta tag </body>.

Entendendo a diferença entre listas ordenadas e não ordenadas

  1. LISTAS ORDENADAS: Listas ordenadas são declaradas para criar listas com números, letras ou algarismos romanos, é obrigatório a abertura <ol> e o fechamento desta tag </ol>, e cada item da lista é criada com a abertura <li> e o fechamento da tag <li>.
  2. LISTAS NÃO ORDENADAS: Listas não ordenadas são declaradas para criar listas com marcadores, é obrigatório a abertura <ol> e o fechamento desta tag </ol>, e cada item da lista é criada com a abertura <li> e o fechamento da tag <li>.

Links de referências

Links são declaradas para criar hyperlinks entre dados da página e entre outras páginas, tanto em páginas internas como páginas externas, é obrigatório a abertura <a> e o fechamento desta tag </a>, e na abertura é inserido o parâmetro href="" onde é especificado o direcionamento.
